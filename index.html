Open Graph Protocol

The Open Graph Protocol (OGP) is a set of meta tags that website developers can include in their HTML documents to control how content appears when shared on social media platforms. Developed by Facebook, the protocol aims to standardize the representation of web pages on social media by providing a structured way to define metadata.

The Open Graph Protocol enables you to control how your content appears when shared on social media platforms. It's a set of meta tags that define your content's title, description, image, and more.

Key Concepts:

  • og:title: Sets the title of your content.
    • <meta property="og:title" content="Your Content Title">
  • og:description: Defines the description of your content.
    • <meta property="og:description" content="Your content description goes here.">
  • og:image: Specifies the image associated with your content.
    • <meta property="og:image" content="your-image-url.jpg">
  • og:url: Indicates the canonical URL of your content.
    • <meta property="og:url" content="your-website-url">
  • og:type: Describes the type of your content (e.g., website, article).
    • <meta property="og:type" content="website">
